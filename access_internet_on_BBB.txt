Internet Access on BBB
PART I) Configurations OF HOST Machine
1. open terminal -> ip a (for interface names)
	NAME STARTING FROM W  is host's wifi interface and with enxb it is interface of BBB
Note the Interface names 
2.Enable IP forwarding from Host using:
	sudo sysctl net.ipv4.ip_forward=1
3. Enable NAT(Network Address Translation) using:
	sudo iptables --table nat --append POSTROUTING --out-interface <Host_wifi_interface_name> -j MASQUERADE
	sudo iptables --append FORWARD --in-interface <BBB_interface_name> -j ACCEPT
	
PART II) Configurations on BBB
1. Add the default gateway for internet traffic (Host works as a Gateway)
	sudo route add default gw 192.168.7.1
2. Add DNS server
	cd /etc
	sudo nano resolv.conf 
    In this file write:
    	nameserver 8.8.8.8
    Save and close the file.
3. Ping to Google server to check the connection 
	ping google.com
